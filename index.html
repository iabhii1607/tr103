Full Stack Development - 1 Month Training Diary

Week 1: Frontend Basics and Styling

Day 1 - HTML Basics

Learned the fundamentals of HTML. Created basic page structures with elements like headings, paragraphs, lists, links, and images.

Day 2 - Semantic HTML

Focused on semantic HTML5 tags (header, footer, article, section, etc.). Built a sample webpage using semantic tags to enhance accessibility.

Day 3 - Introduction to CSS

Explored CSS basics: selectors, properties, and values. Practiced styling text, colors, borders, margins, and padding.

Day 4 - CSS Box Model and Layout

Studied the box model, inline vs. block elements, and used layout properties like float and position.

Day 5 - Introduction to Tailwind CSS

Started using Tailwind's utility-first classes to quickly style a webpage. Practiced responsive design with breakpoints.

Day 6 - Responsive Design Techniques

Implemented responsive design using Tailwind. Ensured the webpage adapts to different screen sizes.

Day 7 - Bootstrap Basics

Learned the Bootstrap grid system and components. Created a responsive webpage using Bootstrap.

Week 2: React Development

Day 8 - Introduction to React

Set up a React project and learned about JSX. Created functional components and rendered them in the UI.

Day 9 - React State and Props

Worked with state and props in React components. Built simple components that manage and pass data.

Day 10 - React Event Handling

Learned how to handle events in React. Added dynamic behavior to components using event listeners.

Day 11 - React Lifecycle Methods and Hooks

Studied React lifecycle methods and hooks like useState and useEffect for managing state and side effects in functional components.

Day 12 - React Router Basics

Learned to navigate between different views in a React app using React Router. Implemented basic routing for a multi-page app.

Day 13 - State Management with Redux

Explored state management in React using Redux. Set up a simple global state management system in a small project.

Day 14 - Project Setup: Finance Tracker

Started a small project to build a Finance Tracker in React. Focused on breaking the UI into components and managing the app state with hooks.

Week 3: Advanced React and Minimal Backend

Day 15 - Advanced React Concepts

Learned about higher-order components and React context for passing data through the component tree.

Day 16 - Introduction to Node.js

Briefly covered Node.js and its use in backend development. Understood the basics of setting up a Node.js server.

Day 17 - Backend with Express.js (Minimal)

Explored Express.js to create basic routes and handle API requests. Implemented a simple backend to serve data for the Finance Tracker.

Day 18 - Integrating Backend with React

Connected the React app with the Express backend using fetch API. Rendered data from the backend in the frontend.

Day 19 - Basic Authentication in React

Implemented a simple login system on the React app using local state to manage user authentication.

Day 20 - Project Update: Finance Tracker with Backend Integration

Enhanced the React Finance Tracker app with backend integration. Added functionality to save income and expense records on the server.

Day 21 - Debugging and Refactoring

Debugged the React project, fixed issues with state management, and refactored code for better maintainability.

Week 4: Project Completion

Day 22 - Finalizing Finance Tracker

Added final features to the Finance Tracker app, including UI improvements and performance optimizations.

Day 23 - Testing and Debugging

Performed testing of the app to identify and fix any remaining bugs. Ensured smooth user experience across different browsers.

Day 24 - Deploying Finance Tracker

Deployed the React Finance Tracker app to a hosting platform. Prepared the project for production deployment.

Day 25 - Wrap-Up and Review

Reviewed the project and summarized key learnings. Presented the final version of the Finance Tracker app.
